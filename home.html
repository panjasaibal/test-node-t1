üîê Authentication APIs

POST /auth/register

Registers a new user.

Payload:

{
  "username": "alice",
  "password": "password123"
}

Response:

{
  "message": "User registered successfully",
  "userId": "uuid-value"
}

POST /auth/login

Logs the user in and sets a HttpOnly refresh token cookie.

Payload:

{
  "username": "alice",
  "password": "password123"
}

Response:

{
  "accessToken": "your_jwt_token"
}

GET /auth/refresh

Issues a new access token using refresh token cookie.

Response:

{
  "accessToken": "new_access_token"
}

POST /auth/logout

Clears refresh token cookie and logs out user.

{
  "message": "Logged out successfully"
}
